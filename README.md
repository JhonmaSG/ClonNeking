# ClonNeking

El objetivo de este proyecto es realizar un clon basandose en la aplicación bancaria como guía, donde tendra algunas funcionalidades basicas de:

- Login
- Código de seguridad
- Enviar dinero
- Sacar dinero
- Colchon
- Bolsillo
